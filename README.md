# Portfólio de Projetos - Enrico Schultz Breda

## Descrição

Este é meu portfólio online, onde apresento os projetos que desenvolvi durante minha trajetória como estudante de Análise e Desenvolvimento de Sistemas. O objetivo é mostrar meu aprendizado e evolução, com foco em qualidade, usabilidade e organização.

## Tecnologias e Métodos Utilizados

- **HTML5 Semântico**  
  Utilizei HTML5 para estruturar o conteúdo de forma clara e acessível, empregando tags semânticas como `<header>`, `<main>`, `<section>` e `<footer>`. Isso melhora a organização do código, acessibilidade e SEO.

- **Bootstrap 4**  
  Escolhi o Bootstrap 4 para garantir um layout responsivo e atraente, com componentes prontos que facilitam o desenvolvimento. O grid system do Bootstrap adapta o portfólio a diferentes tamanhos de tela, garantindo boa experiência em dispositivos móveis e desktops.

- **CSS Personalizado**  
  Apesar do uso do Bootstrap, criei um arquivo CSS customizado para personalizar detalhes visuais, dando identidade própria ao site.

- **Componentes Visuais**  
  Utilizei cards para apresentar cada projeto, facilitando a visualização das informações e imagens de forma organizada e intuitiva. A navbar com menu colapsável melhora a navegação, especialmente em telas menores.

- **Imagens e Ícones SVG**  
  Imagens ilustrativas tornam o portfólio mais atraente e ajudam a transmitir melhor o contexto dos projetos. Ícones SVG, como o do GitHub, garantem gráficos nítidos e leves.

- **Links Externos para Contato**  
  Integrei links para meu GitHub e LinkedIn, facilitando o acesso a mais informações sobre meu perfil e trabalhos.

## Por que esses métodos?

- A combinação de HTML5 semântico e Bootstrap proporciona uma base sólida, rápida de implementar e de fácil manutenção.
- A responsividade é essencial para oferecer boa experiência em qualquer dispositivo.
- A organização por cards e a navegação intuitiva melhoram a usabilidade.
- O CSS customizado possibilita personalizar o design sem perder a agilidade.
- Usar SVGs para ícones mantém o site leve e visualmente consistente.
